# Jetstream Storybook

Welcome to the Jetstream Storybook, a comprehensive collection of components, patterns, and design documentation for the Jetstream platform.

## Getting Started

Explore the components using the sidebar navigation. Components are organized into the following categories:

- **Core UI**: Basic UI elements like buttons, forms, and cards
- **Features**: Complete features and larger components
- **JetShare**: Components related to the JetShare application
- **Components**: Reusable elements across the platform

## Component Documentation

Each component includes:

- **Overview**: Documentation and explanation
- **Examples**: Interactive instances with configurable props
- **Usage guidelines**: Best practices and implementation details

## Using this Storybook

- **Navigation**: Use the sidebar to browse components
- **Search**: Find components by name or functionality
- **Controls**: Adjust component props to see different states
- **Docs**: View detailed documentation for each component
